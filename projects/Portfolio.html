Portfolio Website

Personal portfolio designed and developed to showcase my projects, skills, and journey in technology.

Overview

This portfolio serves as my digital space on the internet. It highlights my projects, interests, and experiences while providing a simple way for people to learn more about me and my work.

Features

  • Minimal and responsive design
  • Project showcase pages
  • Direct links to GitHub, LinkedIn, and Email
  • Modern typography and clean UI
  • Mobile-friendly layout

Tech Stack

HTML • CSS • JavaScript

Design Goals

The goal was to create a portfolio that feels simple, professional, and easy to navigate. Instead of flashy animations and heavy effects, the focus was placed on readability, content, and user experience.

  • Project organization
  • Website deployment
  • Performance optimization
