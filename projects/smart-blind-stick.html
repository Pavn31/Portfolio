Smart Blind Stick

Arduino-based assistive device for visually impaired individuals.

Overview

Smart Blind Stick is an assistive technology project designed to help visually impaired people navigate safely. The system detects obstacles and environmental hazards using sensors and provides alerts through a buzzer.

Features

  • Obstacle detection using HC-SR04 ultrasonic sensor
  • Water detection for puddles and wet surfaces
  • Audio alerts using buzzer
  • GPS location tracking
  • Emergency communication using GSM module

Tech Stack

Arduino UNO • HC-SR04 • GSM (SIM800L) • GPS (NEO-6M) • Water Sensor • Embedded C

What I Learned

Through this project I gained experience working with sensors, serial communication, embedded programming, GPS systems, GSM communication, and real-world hardware integration.

Future Improvements

  • Voice navigation assistance
  • Mobile application integration
  • Machine learning based obstacle classification
  • Rechargeable battery system
